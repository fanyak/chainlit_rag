# Καλωσήρθατε στο Foros chat! 🤖

Χαίρεται!👋 Το Foros chat είναι ένα ισχυρό εργαλείο που στοχεύει να σας βοηθήσει να βρείτε τις πληροφορίες που αναζητείται στα έγγραφα της ΑΑΔΕ.

## Χρήσιμες Συμβουλές 🔗

-**Να είστε συγκεκριμένοι στα ερωτήματά σας:** Αντί για «πώς φορολογούμαι», ρωτήστε «πώς φορολογούνται τα εισοδήματα από ενοίκια για το 2024».

-**Κάντε μία ερώτηση τη φορά:** Για πιο ακριβείς απαντήσεις, αποφύγετε τις πολλαπλές ερωτήσεις σε ένα μήνυμα.

Μην ξοδεύετε τα token σας για να κάνετε ερωτήσεις που δεν αφορούν στην ελληνική νομοθεσία. Το μοντέλο AI έχει οδηγίες να απαντάει μόνο σε φορολογικά ερωτήματα!

-**Ζητήστε διευκρινίσεις:** Αν η απάντηση δεν είναι πλήρης, μη διστάσετε να ρωτήσετε για περισσότερες λεπτομέρειες.

## Σημαντική Σημείωση

Ο Φορολογικός Βοηθός παρέχει πληροφορίες για ενημερωτικούς σκοπούς και **δεν υποκαθιστά** τις υπηρεσίες εξειδικευμένου λογιστή ή φοροτεχνικού. Για πολύπλοκες φορολογικές υποθέσεις ή επίσημες δηλώσεις, συνιστούμε να συμβουλευτείτε επαγγελματία.
